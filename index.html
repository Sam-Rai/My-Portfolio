BoomBikes sharing multiple linear regression

Created a Multiple linear regression model with Training accuracy - 82.3% and Testing accuracy - 78.5% on a US bike-sharing provider BoomBikes dataset which recently got suffered considerable dips in their revenues due to the ongoing Corona pandemic to find out the demand for shared bikes once pandemic is ended with the available independent variables which can be used by the management to understand how exactly the demands vary with different features. Steps performed - Exploratory Analysis, Data Preparation - Dummy variables/Mapped Variables, Test train split, Feature Scaling, Model building - Features selection , Handling multicollinearity, Model Evaluation - Residual Analysis , Predictions , Finding result.
